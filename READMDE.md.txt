# ETL Supermarket Sales

Projeto simples de ETL usando Python, Pandas e SQLite.

## Objetivo
- Ler CSV com dados de vendas
- Transformar os dados (seleção de colunas e cálculo de média)
- Salvar em banco SQLite

## Tecnologias
- Python
- Pandas
- SQLite
- Jupyter Notebook

## Como rodar
1. Baixar o CSV: supermarket_sales.csv
2. Abrir o notebook `ETL_Supermarket.ipynb` e rodar as células
